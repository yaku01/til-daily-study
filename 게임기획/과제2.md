**목표** : 실제 게임의 특정 시스템을 3단계 역기획서 구조로 문서화한다.

**분석 대상 선택 (하나 선택)**

| 선택 | 대상 |
| --- | --- |
| A | 원신 — 원소 반응 시스템 |
| B | 리그 오브 레전드 — 시야(와드) 시스템 |
| C | 엘든링 — 스태미나 시스템 |
| D | 클래시 로얄 — 트로피 / 덱 시스템 |
| E | 스타듀밸리 — NPC 호감도 시스템 |
| F | 본인 선정 (시스템 하나로 한정) |

**작성 구조**

1. **정의** : 공식 명칭과 게임 내 위치 / 게임 전체에서의 역할 / 유사 시스템 비교 1~2개
2. **구성** : 시스템 구성 요소 전체 나열 / 수치와 조건 (출처 명시) / 스크린샷 2장 이상
3. **상세** : 정상 동작 흐름 / 예외 케이스 2가지 이상 / 기획 의도 추론 300자 이상 (근거 포함)
4. **비판적 평가** : 기획이 성공한 부분 1개 / 아쉬운 부분 1개 / 내가 기획자라면 어떻게 수정할지 제안

**주의사항**

- 단순 공략 수준 서술 금지
- 모든 수치에 출처 명시 (직접 측정 / 공식 패치노트 / 커뮤니티 검증)
- "재미있다", "불편하다" 등 감상 서술 금지 — 메커니즘 기반 근거 필수
